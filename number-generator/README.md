# Number-generator

## Описание:
Простое приложение "Генератор случайных чисел". В приложении по умолчанию выставлены границы генерации от 0 до 100. Пользователь может вносить изменения в пределы генерации числа.

## Стэк:
- UIKit
- MVC
- Label
- View
- верстка кодом
